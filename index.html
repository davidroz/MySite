Q1:   What is the different between web1.0, web2.0 and web3.0 ?

Web 1.0
Simply an information portal where users passively receive information without the ability to post any feedback as comments and reviews.
This version is very limited in interaction between sites and web-users.
It is often called as the "readable" phrase of the web with flat data.

Web 2.0
Unlike Web 1.0, Web 2.0 facilitates interaction between web users and sites, so it allows users to interact more freely with each other.
This version encourages participation, collaboration, and information sharing.
It is often called as the “writable” phrase of the web world with interactive data.
Examples of Web 2.0 applications are Youtube, Facebook, and so on.

Web 3.0
Web 3.0 is a semantic web which refers to the future. In Web 3.0, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of users. One example of Web 3.0 is Tivo, a digital video recorder. Its recording program can search the web and read what it finds to you based on your preferences.
It is often called the “executable” phrase of web world with dynamic applications, interactive services, and “machine-to-machine” interaction.
